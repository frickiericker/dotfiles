Dotfiles
========

## Usage

Clone this repository into ~/etc and create links:

    git clone https://github.com/frickiericker/dotfiles ~/etc
    ln -s ~/etc/.zshenv    ~/.zshenv
    ln -s ~/etc/.zshrc     ~/.zshrc
    ln -s ~/etc/.tmux.conf ~/.tmux.conf
    ln -s ~/etc/.vim       ~/.vim
    ln -s ~/etc/.vim       ~/.config/nvim
